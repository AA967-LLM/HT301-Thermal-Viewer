GUI Features & Capabilities

The frontend was completely rebuilt using PyQt6 to support High-DPI displays with a glassmorphic aesthetic.

  • Live Multi-ROI Engine: Click and drag directly on the video feed to draw Points, Lines, Rectangles, and Polygons. Each shape calculates real-time Minimum, Maximum, Average, and Standard Deviation temperatures.
  • Dynamic Radiometric Parameters: Update Emissivity, Ambient Temperature, and Object Distance on the fly in the left-hand dock. The entire thermal span updates instantly based on real physics formulas.
  • Isotherm & Moisture Overlays: Automatically highlight bands of temperature matching your criteria. Used in HVAC for pinpointing dew points or in electrical for flagging breakers over 60°C.
  • Customizable Export Path: A dedicated Capture tab lets you browse and set your output directory. Saves automatically sync to Pictures\Thermal Camera\YYYY-MM-DD\.
  • Smart Layouts: ScrollAreas prevent layout collapse on 150% scaled 2560x1600 laptop displays. Fonts are mapped in absolute `pt` units, not relative pixels, preventing Qt scaling bugs.

Step-by-Step Installation

The app is explicitly designed to be lightweight. It does not require multi-gigabyte CUDA/PyTorch installations, running flawlessly on pure CPU via highly optimized NumPy operations.

  1. Clone the Repository

    Open your terminal or command prompt (Windows/Linux) and clone the source code to your machine.

    git clone https://github.com/AA967-LLM/HT301-Thermal-Viewer.git
cd HT301-Thermal-Viewer
  2. Create the Virtual Environment

    Isolate your dependencies to prevent conflicts with other Python projects.

    python -m venv .venv
  3. Activate the Environment

    On Windows (Command Prompt or PowerShell):

    .venv\Scripts\activate

    On Linux / macOS:

    source .venv/bin/activate
  4. Install Dependencies

    Install PyQt6, OpenCV, and NumPy from the requirements file. This should download roughly ~100MB of dependencies.

    pip install -r requirements.txt
  5. Launch the Application

    Ensure the HT-301 camera is plugged into a USB 3.0 port before launching.

    python app/main.py

How to Run & Use the Application

  1. Connecting the Camera

    Plug the HT-301 into your USB port using a reliable USB-C to USB-A/C cable. Launch the app using python app/main.py. The application automatically probes DirectShow/V4L2 for the HT-301 hardware ID and initiates the 25Hz stream.

  2. Adjusting Palettes & Range

    In the Radiometric Params panel on the left, select your preferred color palette (e.g., Iron for general use, White Hot for security). Toggle "Manual Range" to lock the Min/Max temperature scale if you are tracking a specific heat source.

  3. Drawing Regions of Interest (ROI)

    Click the "Add Rectangle" or "Add Line" button in the toolbar. Click and drag across the live video feed. A neon bounding box will appear, displaying live Min/Max/Avg temperatures specifically for those pixels.

  4. Capturing Data

    Click the large Snapshot button or use the Spacebar hotkey. The app triggers the Auto-Exporter.

    Check your Pictures\Thermal Camera folder. You will find:

    • image_YYYYMMDD_HHMMSS.png (The visual image)
    • data_YYYYMMDD_HHMMSS.npy (The raw float32 temperature array for post-processing)
    • meta_YYYYMMDD_HHMMSS.json (Emissivity, distance, and environmental constants)
